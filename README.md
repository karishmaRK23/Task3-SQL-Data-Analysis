# Task3-SQL-Data-Analysis
## Objective
Analyze the Sample Superstore dataset using SQL to extract business insights.

## Tools Used
- SQLite
- DB Browser for SQLite

## Dataset
Sample Superstore

## SQL Concepts Demonstrated
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (SUM, AVG, MAX)
- Subqueries
- Views
- Indexes

## Key Insights
- Identified high-value sales transactions.
- Compared total sales across categories.
- Calculated regional profits.
- Determined average and maximum sales.
- Created reusable views for high-profit orders.
- Improved query performance using indexes.
